# TomoList

### Social Media, DAW Project

---

## Features

- Sign up / sign in  
- User home page and profile page  
- Publish and manage posts  
- Search for friends, add and contact them  
- Interact with friends’ posts  
- Use a simulated chat system (not WebSockets)  
- See error pages for invalid routes  
- Admin controls for management  

> ⚠️ Note: Some of the PDF "organization" data is fictional due to school project requirements.
