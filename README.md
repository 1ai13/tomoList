# TomoList

## Social Media, DAW proyect

Features:
- User sing up/in
- Home page
- Profile page
- Publish your posts and manage them
- Search your friends and add/contact them
- Interact with your friend's posts
- Chatter (Not WebSockets, faking live chat)
- Error page
- Admin controls
<br>
BackEnd -> Springboot, Java, MySQL, SQL

FrontEnd -> HTML, CSS, JS, Bootstrap, Thymeleaf, jQuery
<br>

Currently not deployed in any domain, access the README.pdf if you want to see partial visibility of the App FrontEnd

Please take in mind the PDF is fully written in Spanish.

Some information like organization structure and that kind of stuff were invented, its fake info due to proyect requirements.
